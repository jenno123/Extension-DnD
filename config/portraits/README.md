# Portraits

Drop your character portrait PNGs here. Filenames must match the "portrait"
values in ../campaign.json.

Recommendations:
- Transparent-background PNG, portrait orientation (e.g. 512x768 or 600x900).
- Keep each file under ~1 MB so the overlay loads fast for every player.
- A bust / upper-body framing reads best in the visual-novel style.

The relay serves these at http://<relay-host>:<port>/portraits/<filename>.
